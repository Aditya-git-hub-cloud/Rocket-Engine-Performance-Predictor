# 🚀 Rocket Propulsion Performance Prediction using Machine Learning

## 📌 Project Overview
This project predicts the **Specific Impulse (Isp)** of a rocket engine using **Machine Learning (Linear Regression)**.

Specific Impulse is one of the most important parameters in **rocket propulsion systems** because it measures the **efficiency of a rocket engine**.

The goal of this project is to build a **machine learning model that predicts rocket engine performance** based on different propulsion parameters.

The project includes:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation
- Model Saving using Pickle
- Deployment using Streamlit Web App

---

# 📊 Dataset Information

The dataset contains different rocket propulsion parameters used to predict **Specific Impulse (Isp)**.

### Features

- fuel_type
- oxidizer_type
- chamber_pressure
- oxidizer_fuel_ratio
- combustion_temperature
- heat_capacity_ratio
- nozzle_expansion_ratio
- ambient_pressure
- combustion_stability_margin

### Target Variable

**Specific Impulse (Isp)**
